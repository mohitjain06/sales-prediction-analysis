# Sales Prediction and Data Analysis Project

This project focuses on sales prediction and business data analysis using Machine Learning techniques in Python.

The project analyzes retail sales data, visualizes business insights, and predicts sales using different regression models.

---

## Features

- Sales Data Analysis
- Monthly Sales Trend Analysis
- City-wise Sales Analysis
- Product Performance Analysis
- Category-wise Sales Distribution
- Daily Sales Trend Visualization
- Shipping Time Analysis
- Machine Learning Based Sales Prediction
- Feature Importance Visualization

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost

---

## Machine Learning Models Used

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

---

## Dataset Information

The project is trained on a retail sales dataset containing information such as:

- Product Name
- Sales
- Order Date
- Ship Date
- Category
- Sub-Category
- Region
- Segment
- City
- Ship Mode
